Netzwerk-Funktionen

In dieser Datei werden einige Netzwerk-Funktionen definiert.

char *readline(int sock,rline_t *tsd);

Diese Funktion liest eine Zeile von einem Socket ein. Sie ist thread-safe. Sie erwartet als ersten Parameter den Socket und als zweiten Parameter eine rline_t-Struktur. Achtung! Diese Funktion alloziiert Speicher, den man mit free() wieder freigeben muss! Beispiel: 

rline_t *tsd = malloc(sizeof(tsd));
char *line;
bzero(tsd,sizeof(tsd));

line = readline(sock,tsd);
if(line) {
  printf("%s",line);
  free(line);
}

free(tsd);

ssize_t writen(int sock,const void *buff,size_t n);

Diese Funktion schreibt n Bytes in den Socket sock aus dem Buffer buff.

static ssize_t my_read(rline_t *tsd,int fd,char *ptr)

Diese Funktion liest MAXLINE Zeichen von dem File-Descriptor fd ein und Puffert sie in tsd. Das erste Zeichen aus dem Puffer wird in ptr gespeichert.
