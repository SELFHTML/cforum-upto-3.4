Standard-Definitionen

In dieser Datei werden einige Standard-Definitonen gemacht:

  • ARG_IGNORE Siehe Konfigurations-Parser
  • ARG_NONE Siehe Konfigurations-Parser
  • ARG_INT Siehe Konfigurations-Parser
  • ARG_STR Siehe Konfigurations-Parser
  • ARG_YESNO Siehe Konfigurations-Parser
  • ARG_LONG Siehe Konfigurations-Parser
  • PRERESERVE Konstante, die definiert, wieviele Array-Elemente vorreserviert werden sollen.
  • MAXLINE Konstante, die eine Art Limit für Zeilen-Längen definiert.
  • ULL Eine Abkürzung für unsigned long long
  • FO_CONF_OK OK-Wert
  • FO_CONF_PARENTHESIS Klammer- oder Redezeichen-Fehler
  • FO_CONF_WRONG_VAL Falscher Wert
  • E_NO_SOCK Es konnte kein Socket alloziiert werden
  • E_NO_CONN Es konnte keine Verbindung hergestellt werden
  • E_TPL_NOT_FOUND Es konnte ein Template nicht gefunden werden
  • E_NO_THREADLIST Die Threadliste konnte nicht angefordert werden
  • E_DATA_FAILURE Daten-Fehler
  • E_NO_MESSAGE Die Nachricht wurde nicht gefunden
  • LISTENQ Maximale Anzahl an Clients
  • dprintf Wenn DEBUG definiert ist, wird dprintf durch printf ersetzt, ansonsten durch (void).
