Character-Konvertierungs-Funktionen

In dieser Datei befinden sich einige Funktionen, die das Arbeiten mit Character-Sets und die HTML-Ausgabe vereinfachen.

char *utf8_to_iso_8859_1(const char *toencode)

Mit dieser Funktion kann man UTF8-Strings umwandeln nach ISO-8859-1. Es wird intern ein neuer String alloziiert, den der Programmierer per free() wieder freigeben muss! Beispiel: 

char *str = utf8_to_iso_8859_1(utf8string);
if(str) {
  printf("%s\n",str);
  free(str);
}

char *iso_8859_1_to_utf8(const char *toencode)

Mit dieser Funktion kann man ISO-8859-1 wieder zurück wandeln nach UTF8. Auch hier wird intern Speicher reserviert, den man per free() wieder freigeben muss. Beispiel: 

char *str = iso_8859_1_to_utf8(iso88591string);
if(str) {
  printf("%s\n"str);
  free(str);
}

char *htmlentities(const char *string)

Mit dieser Funktion kann man die HTML-relevanten Zeichen umwandeln in Entities. Wiederum wird hier intern Speicher reserviert, den man mit free() wieder freigeben muss. Beispiel: 

char *str = htmlentities(somestring);
if(str) {
  printf("%s\n",str);
  free(str);
}
